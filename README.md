### 🎙️ VoiceScript — AI Speech-to-Text Web App

[![Live Demo](https://img.shields.io/badge/🌐_Live_Demo-00C7B7?style=for-the-badge)](https://speech-to-text-ai-dipan.netlify.app/)
[![GitHub](https://img.shields.io/badge/GitHub_Repo-181717?style=for-the-badge&logo=github)](https://github.com/dipan-cse)

</div>

> **Production-grade MERN Stack application** — built and deployed solo in **14 days**

| Feature | Details |
|---------|---------|
| 🎤 **Live Recording** | MediaRecorder API with pause/resume/stop controls |
| 📁 **File Upload** | Drag & drop support for MP3, WAV, M4A, WebM (up to 50 MB) |
| 🤖 **AI Transcription** | Deepgram nova-2 + OpenAI Whisper (pluggable via `.env`) |
| 🔌 **REST API** | 6 endpoints — transcribe, CRUD, stats, health check |
| 💾 **Database** | MongoDB + Mongoose with word-count, confidence score, pagination |
| 🚀 **CI/CD Deploy** | Netlify (frontend) + Render (backend) with full pipeline |

**Stack:** `React.js` · `Node.js` · `Express.js` · `MongoDB` · `Deepgram API` · `Tailwind CSS` · `Netlify` · `Render`
